## Descrição
Visualização de dados e linguagem SQL: da construção de dashboards (Looker Studio) à consulta e manipulação de dados com SQL.

## O que a matéria realmente ensina
Uso do Looker Studio para conectar fontes, montar visualizações e entregar relatórios. Transição para SQL: consultas (SELECT), filtros, agregações e junções. Qualidade e origem dos dados. Regras de avaliação (TP/AT), rubricas e formato de entrega (PDF, links).

## Para que serve na formação
Prepara para ler, transformar e apresentar dados em contexto profissional. Base para disciplinas de projeto e análise de dados.
