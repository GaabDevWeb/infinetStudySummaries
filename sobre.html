Sobre o projeto

Este é um site educacional estático que transforma transcrições de aulas em material de estudo estruturado e consultável.

O objetivo é simples: reduzir o tempo de revisão e o esforço de extração de conteúdo. Em vez de reassistir aulas longas ou reler transcrições extensas, você acessa resumos padronizados, organizados por disciplina e por aula.

O foco é estudo operacional, localizar conceitos, revisar rapidamente e consolidar entendimento.

Problema que ele resolve

Aulas gravadas e transcrições completas são úteis, mas pouco eficientes para revisão. Encontrar pontos específicos exige tempo e atenção desnecessários.

Este projeto reduz custo cognitivo ao converter material bruto em resumos estruturados, com seções fixas, conceitos-chave e organização consistente.

Como funciona

Fluxo de processamento:

  • Aula gravada (vídeo ou áudio)
  • Transcrição completa
  • Processamento com prompt padronizado
  • Geração de resumo estruturado em Markdown
  • Renderização no site com HTML, CSS e JavaScript

Sem banco de dados. Sem backend. Conteúdo versionado.

Repositório e código

O projeto é mantido em repositório público no GitHub. Código, estrutura e resumos em Markdown vivem no mesmo lugar e são versionados.

O site é publicado via GitHub Pages a partir desse repositório.

Como contribuir

Você pode contribuir de três formas objetivas:

  • corrigindo erros técnicos nos resumos
  • melhorando clareza ou organização de seções
  • adicionando novos resumos no formato padrão do projeto

As contribuições são feitas via Pull Request com arquivos .md, seguindo o template estrutural existente.
